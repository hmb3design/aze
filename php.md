# Fiche de révision sur le PHP

## Fonctions
Une fonction sert a créer une fonctionnalité.


En PHP, on crée une fonction comme cela:
```php
<?php
function nom_de_la_fonction ($parametres) {
  // ... code
}
```

## Variables
On crée une variables comme cela:
```php
<?php
$nom_variable = "texte";
$nom_variable_2 = 12;
```
